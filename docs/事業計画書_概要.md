## サービス概要

**サービス名**：BRIDGE（ブリッジ）

Claude・GPT・Geminiが互いの回答を読み、議論し合い、統合された最終回答を導き出すプラットフォーム。

### 競合との差別化

| | Chat Hub（競合） | BRIDGE |
|---|---|---|
| 仕組み | 回答を横並び表示するだけ | AIが互いに議論して統合回答を生成 |
| 体験 | 並列表示 | 対話・成長 |

### 議論フロー

1. ユーザーが質問を入力（普通に話しかけるだけ）
2. BRIDGEが裏でプロンプトを最適化
3. GPT → 仮説・選択肢を発散
4. Claude → 批判・深掘り・見落とし指摘
5. Gemini → ベストDoを統合・結論
6. 統合回答を表示

## セキュリティ設計

**プライバシーファースト：データを持たない**

- BRIDGEサーバー：保存ゼロ
- APIキー：ユーザーのブラウザ内にのみ存在（サーバー経由なし）
- 各AIのAPI：学習データに使われない

「あなたの会話はBRIDGEのサーバーに一切保存されません。APIキーも私たちは保持しません。」
